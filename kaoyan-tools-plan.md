# PROGRESS:进展
# 2026 考研效率工具开发计划

## 目标
基于你熟悉的 Java + SpringBoot 技术栈，搭建一套服务于 2026 考研（英一、数一、政治、408）的个人学习工具，替代 Excel/纸质错题本，提高复习与记忆效率。

## 技术栈
- 后端：SpringBoot 3.x + Spring Data JPA
- 数据库：H2（本地开发，开箱即用）
- 前端：Thymeleaf（最小可用界面，快速验证）
- 构建：Maven
- 部署：本地 Jar 启动，数据随项目走

## 功能方向

### 1. 记忆卡片系统
- 卡片字段：问题、答案、科目、标签、难度、创建时间
- 复习算法：艾宾浩斯遗忘曲线 / 自定义间隔
- 每日复习队列：只显示当天该复习的卡片
- 统计：已掌握、学习中、待复习数量

### 2. 错题本
- 错题字段：题目描述、正确答案、错误原因、所属科目/章节/知识点、是否已掌握
- 筛选：按科目、知识点、错误原因、掌握状态
- 随机抽题重做：从错题库中按权重抽取
- 薄弱知识点统计：错误次数 Top N

### 3. 每日计划与专注计时
- 每日任务：科目、内容、预计时长、优先级、完成状态
- 番茄钟：25 分钟专注 + 5 分钟休息
- 学习时长记录：每个科目实际耗时
- 周/月统计报表

### 4. 408 知识图谱
- 知识点树：数据结构、计算机组成原理、操作系统、计算机网络
- 题目与知识点关联：错题/卡片可绑定多个知识点
- 掌握度热力图：按知识点统计正确率
- 复习路径推荐：优先推荐薄弱知识点相关题目

## 阶段规划

| 阶段 | 内容 | 预计时间 |
|---|---|---|
| 阶段一 | 项目脚手架 + 公共模块（科目、知识点、统一响应） | 1 天 |
| 阶段二 | 记忆卡片系统 MVP | 1-2 天 |
| 阶段三 | 错题本 + 408 知识点关联 | 2-3 天 |
| 阶段四 | 每日计划 + 番茄钟 + 统计 | 2-3 天 |
| 阶段五 | 408 知识图谱 + 可视化 | 3-5 天 |
| 阶段六 | 打包、本地运行优化、数据导入导出 | 1-2 天 |

## 数据模型初稿

- `Subject`：科目（英一、数一、政治、408 及子科目）
- `KnowledgePoint`：知识点，支持树形结构
- `Card`：记忆卡片
- `Mistake`：错题
- `DailyTask`：每日任务
- `FocusSession`：专注记录
- `ReviewRecord`：复习记录（卡片/错题的每次复习结果）
